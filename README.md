## 👋 About

This portfolio website is built as a personal platform to showcase my background, skills, experience, and projects as a Full-Stack Software Engineer.

In addition to frontend development, this project focuses heavily on real-world deployment practices, including cloud infrastructure setup, domain configuration, and production server management.

---

## ⭐ Highlights

- Fully self-hosted production environment  
- Oracle Cloud Infrastructure (OCI) virtual machine  
- Custom purchased domain (jackyong.dev)  
- Manual Nginx reverse proxy configuration  
- PM2 process management for Node.js applications  
- No managed deployment platforms used  

This project reflects real engineering workflows used in professional environments.

---

## 🧠 Features

- Responsive modern UI  
- Multilingual routing (example: /home/en)  
- Project and experience showcase  
- SCSS-based custom styling system  
- Clean, modular, and maintainable architecture  

---

## 🛠️ Tech Stack

**Frontend:** Next.js, TypeScript, SCSS  

**Infrastructure & Deployment:** Oracle Cloud Infrastructure (OCI), Linux, Nginx, PM2, Custom Domain & DNS Configuration
